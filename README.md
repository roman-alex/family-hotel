# FAMILY HOTEL

Односторінковий сайт бази відпочинку «FAMILY HOTEL» (React + Vite + TypeScript + Tailwind).

## Розробка

```bash
npm install
npm run dev
```

## Збірка

```bash
npm run build
npm run preview
```

## Деплой

GitHub Pages: `https://roman-alex.github.io/family-hotel/`

Контент редагується в `src/data/content.ts`.
